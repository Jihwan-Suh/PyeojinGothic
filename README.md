## 웹폰트
#### HTML
```Ruby
<link rel="stylesheet" as="style" crossorigin href="https://cdn.jsdelivr.net/gh/Jihwan-Suh/PyeojinGothic/CSS/pyeojin-font.css"/>
```
#### CSS
```Ruby
@import url("https://cdn.jsdelivr.net/gh/Jihwan-Suh/PyeojinGothic/CSS/pyeojin-font.css");
```
